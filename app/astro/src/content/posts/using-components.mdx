Astroコンポーネント内のフロントマターでインポートを行う。

## 基本コンポーネント

### Badge

```mdx
<Badge>Badge</Badge>
<Badge colorPalette="primary">Primary</Badge>
<Badge colorPalette="red" size="md">Red</Badge>
```

<Badge>Badge</Badge> <Badge colorPalette="primary">Primary</Badge> <Badge colorPalette="red" size="md">Red</Badge>

### Button

```mdx
<Button>Button</Button>
<Button colorPalette="primary">Primary</Button>
<Button size="md" colorPalette="secondary" visual="outline">
  Outline
</Button>
<Button size="lg" icon="mdi:arrow-right-bold-circle" iconPlace="end">
  With Icon
</Button>
```

<Button>Button</Button> <Button colorPalette="primary">Primary</Button> <Button size="md" colorPalette="secondary" visual="outline">Outline</Button> <Button size="lg" icon="mdi:arrow-right-bold-circle" iconPlace="end">With Icon</Button>
